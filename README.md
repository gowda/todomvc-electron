# todomvc-electron

Implementation of [TodoMVC application spec](https://github.com/tastejs/todomvc/blob/master/app-spec.md) using `electron` in `TypeScript`.

UI is implemented using `React` in TypeScript.

## Running
```bash
$ npm run start
```

## Tests
```bash
$ npm run test
```

#### Feature tests
```bash
$ npm run test-features
```

#### Unit tests
```bash
$ npm run test-unit
```
